# Design Artifacts

This folder contains the Phase 2 design inputs that guide implementation.

- `Project_Phase_2_Group4_Report_Detailed_Design.pdf`: detailed design report and source blueprint.
- `Project_Phase_2_Group4_Presentation.pdf`: Phase 2 presentation reference for continuity checks.
- `Smart_City_Architecture_Group4_Final.html`: companion architecture diagram.

Implementation should follow these documents while using practical engineering adjustments such as local-first development, adapters for cloud services, CI, and reproducible demo data.
